Automação Facebook / Instagram

Preencha API e conta, envie os vídeos e defina textos e orçamento. Será criada uma campanha pausada (para você conferir antes de ligar). Conjuntos e anúncios ficam ativos dentro da conta; para veicular, basta ativar a campanha no Gerenciador.

Conta e API
Configuração da campanha

Em CBO, você define o orçamento no nível da campanha e pode criar múltiplos conjuntos com segmentações diferentes.

Em ABO, o orçamento é definido dentro de cada conjunto.

Personalizar conjuntos
Conjuntos (vale para CBO e ABO)
Cada conjunto tem nome, direcionamento detalhado e vídeos próprios.

Com ABO, cada conjunto usa orçamento próprio com estratégia de maior volume.
